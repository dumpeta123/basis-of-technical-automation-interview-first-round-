# Automation Project

This project contains automated tests using Selenium WebDriver, Java tools. The tests are designed to verify various functionalities of the application under test.

## Prerequisites

Before running the tests, ensure you have the following installed:

- Java Development Kit (JDK) 1.8 or higher
- Maven
- IntelliJ IDEA or any other IDE
- Git

## Setup


1.**Clone the repository:**

```bash
git clone <repository_url>
```

2.**Navigate to the project directory:**


```bash
cd <project_directory>
```

3.**Install dependencies:**

```bash
mvn clean install
```

## Running Tests

1.**Using IntelliJ IDEA**
- Open the project in IntelliJ IDEA.

2.**Build the project:**

- Go to Build > Build Project or press Ctrl+F9.

3.**Run the tests:**

- Navigate to the src/test/java/TestCases directory.
- Right-click on the TestCase file and select Run.
